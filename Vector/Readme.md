## **Vector** is a class that creates a dynamic array allowing insertions and deletions at the back.

First import Vector Library
```cpp
#include <vector>
```

#### Syntax 
  - 1D Vector
  ```cpp
  // vector  <DataType>  VectorName(VectorSize);
  vector  <int>  v(6);
  ```
  - 2D Vector
  ```cpp
  vector  <vector  <int>>  vec;
  ```
